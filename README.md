# ⚖️ Bricks-to-Weight | Bricks-Repo Tool

[EN] A simple, fast, and mobile-friendly tool to estimate the number of LEGO® pieces based on weight.
[UA] Простий та швидкий інструмент для оцінки кількості деталей LEGO® за їхньою вагою.

### Features / Особливості:
- No registration / Без реєстрації
- Multilingual (EN/UA) / Двомовний
- Fast calculation (1.5g/pc ratio) / Швидкий розрахунок
